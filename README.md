# 📘 E-Rapot  
**E-Rapot** adalah aplikasi **sistem rapor elektronik berbasis web** yang dibangun menggunakan **framework Laravel** (PHP) dan database untuk membantu sekolah dalam mengelola nilai siswa. Aplikasi ini memberikan kemudahan dalam input nilai, cetak rapor, serta manajemen data akademik secara online.

---

## 🛠️ Teknologi  
- 🌀 Laravel (PHP Framework)  
- 💾 Database (MySQL / PostgreSQL)  
- 🧠 MVC Architecture  
- 🧩 Frontend Blade Templates

---

## ⭐ Fitur Utama  
✨ Autentikasi multi-role  
✨ Manajemen data Siswa, Guru, Kelas  
✨ Input dan penilaian nilai siswa  
✨ Cetak rapor ke PDF  
✨ Pengaturan akses guru & wali kelas
